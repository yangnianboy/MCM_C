# 2025 MCM Problem C: 奥运会奖牌榜模型

除了观看最新一届在巴黎举行的2024年夏季奥运会期间的各项赛事外，粉丝们还关注着每个国家的整体"奖牌榜"。最终结果（表1）显示，美国获得奖牌总数最多（126枚），中国和美国在金牌榜上并列第一（40枚）。东道主法国在金牌榜上排名第5（16枚），但在奖牌总数上排名第4，而英国以14枚金牌排名第7，在奖牌总数上排名第3。

**表1：巴黎奥运会（2024）最终奖牌榜——金牌前7名国家[1]**

奖牌榜顶部的排名总是备受关注，但其他国家的奖牌数往往同样重要。例如，阿尔巴尼亚（2枚奖牌）、佛得角、多米尼加和圣卢西亚（2枚奖牌）在巴黎奥运会上赢得了他们国家的首枚奥运会奖牌。多米尼加和圣卢西亚也各自在这些比赛中获得了一枚金牌。仍有60多个国家尚未获得奥运会奖牌。对最终奖牌数的预测很常见，但通常不是基于历史奖牌数，而是在即将举行的奥运会开始前，当已知当前计划参赛的运动员时（例如：https://www.nielsen.com/news-center/2024/virtual-medal-table-forecast/）。

## 术语表

**国际奥委会（IOC）：** 是负责组织夏季和冬季奥运会的国际、非政府、体育管理机构。IOC最著名的是负责组织夏季和冬季奥运会的组织。

**计划：** 奥运会计划是IOC为每届奥运会制定的所有体育竞赛计划。

**SDE：** 运动、项目或赛事

**运动：** IOC将奥运会运动定义为由单一国际体育联合会（IF）管理的项目。单一运动可能包含一个或多个项目，每个项目是一个或多个赛事的焦点。

**项目：** 包含一个或多个赛事的运动分支。

**赛事：** 项目内的竞赛，产生排名和奖项（例如奖牌）。

2024年巴黎奥运会计划中运动、项目和赛事关系的示例：

• 世界水上运动是管理水上运动运动的IF

• 在水上运动运动中有多个项目——花样游泳、跳水、马拉松游泳、游泳和水球。

• 在跳水项目中有八个奖牌赛事：

  • 个人3米跳板 - 男子和女子

  • 个人10米跳台 - 男子和女子

  • 同步3米跳板 - 男子和女子

  • 同步10米跳台 - 男子和女子
