# 2025 MCM Problem C 任务清单

## 阶段一：数据探索与预处理

- [ ] 读取并理解所有数据文件（5个CSV文件）
- [ ] **建立国家/地区实体映射表**（处理IOC代码变化：苏联→俄罗斯、东德/西德→德国、Germany-1等格式）
- [ ] 数据清洗（处理异常值、缺失值、团队数据格式）
- [ ] 数据探索性分析（历史趋势、初步统计）

## 阶段二：基础分析（东道主效应与项目影响）

- [ ] **量化东道主效应系数**（从历史数据计算：作为东道主平均能多拿多少金牌/总奖牌）
- [ ] **分析项目设置对奖牌数的影响**（各运动项目对各国奖牌贡献率）
- [ ] **分析东道主项目选择的影响**（东道主如何通过项目设置影响自身和其他国家奖牌数）
- [ ] **识别各国优势运动项目**（计算各国在各运动项目的历史表现和权重）

## 阶段三：核心模型开发

- [ ] **特征工程**（**仅使用提供的数据集**，不使用外部GDP/人口数据）
  - [ ] 历史奖牌特征（过去N届平均、趋势、动量）
  - [ ] 东道主特征（使用阶段二计算的系数）
  - [ ] 项目特征（项目总数、各运动项目数量，使用阶段二计算的权重）
  - [ ] 运动员数据特征（参赛人数、项目覆盖率，来自summerOly_athletes.csv）
  - [ ] 时间特征（年份、届数）
  - [ ] **检查点：确认未使用外部数据作为直接特征**
- [ ] 开发金牌数预测模型（尝试多种模型：计数回归、时间序列等，选择最优）
- [ ] 开发总奖牌数预测模型（尝试多种模型，选择最优）
- [ ] 模型验证与不确定性分析（交叉验证、预测区间、精度评估）

## 阶段四：2028年洛杉矶奥运会预测

- [ ] 准备2028年预测数据（项目设置、美国为东道主、各国家特征）

- [ ] 分析预测结果（识别进步/退步国家，分析原因：东道主效应、项目变化等）

## 阶段五：专项攻关

### 5.1 首次获得奖牌国家预测

### 5.2 "伟大教练"效应分析

- [ ] **定义并检测数据中的结构性断点（Structural Breaks）**

- [ ] 量化"伟大教练"效应贡献（效应大小、持续时间、在总奖牌数中的占比）
